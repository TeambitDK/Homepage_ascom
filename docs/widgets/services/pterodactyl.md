[Pterodactyl](https://github.com/pterodactyl) - The free game server control panel for users, networks, and game service providers.

Allowed fields: `["nodes", "servers"]`

```yaml
widget:
  type: pterodactyl
  url: http://pterodactylhost:port
  key: pterodactylapikey
```
