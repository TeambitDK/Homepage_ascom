---

Learn more about [Whats Up Docker](https://github.com/fmartinou/whats-up-docker).

Allowed fields: `["monitoring", "updates"]`.

```yaml
widget:
  type: whatsupdocker
  url: http://whatsupdocker:port
  username: username # optional
  password: password # optional
```
